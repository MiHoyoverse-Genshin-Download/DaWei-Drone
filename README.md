
# Global Drone Remote: Embedding drones with SIM7600 and ESP8266 

this project will use SIM7600 and ESP8266 to mimic cheap drones controller, making a global remote controller for any drone.

using facial recognition to detect DaWei and top Hoyoverse managers either by remote control or autonomous AI driven drones.



## Requirements

- Any cheap toy drone
- SIM7600 and ESP8266 (tx/rx connected + power)
- a(your) payload


## How it works?

using pre trainer facial recognition with basic code/api to control the drone with an app, attach the payload directly under the drone and connect SIM7600+ESP8266 to the drones power supply. 

can be placed anywhere in Xuhui district in ~5km radius on any rooftop, as even the cheapest drone can do a one-way trip for atleast 5km and can be idle for more than a year.




## Features
Phone app, server and hardware code plus Facial recognition models provided and ready to go.
total setup for each drone takes less than 2hours.
